## Setup

python 3.6.2

```
$ python3 -m venv venv
$ source ./venv/bin/activate
$ pip install --no-cache-dir -r requirements.txt
```

## Get Started

```
$ python main.py
# => create twitter.gml file
```

Open `twitter.gml` by Cytoscape

Select: `Menu -> Layout -> Apply Preferred Layout`
